# QIDI Plus4 Mods
## Essential Mods

These are the mods I have added that I consider essential for the printer:

1. [stew675's Beacon3D H Mount](https://www.printables.com/model/1170120-beacon3d-mount-for-qidi-plus4)
2. [MiklasHeinrich's Toolhead Cover](https://www.printables.com/model/1272873-qidi-plus-4-toolhead-cover-for-stew675-beacon) - Essential if you've got a Beacon. I've only printed this in PETG at the moment, as I'm undecided about the toolhead cover situation before I print one in ABS.
3. [stew675's Bed Platform Levelling Blocks](https://www.printables.com/model/1212350-permanent-bed-platform-levelling-blocks-for-the-qi)
4. [stew675's Modular Exhaust System](https://www.printables.com/model/1328634-modular-exhaust-system-for-the-qidi-plus-4) - Did not print the filter cage, as I've printed the HEPA mod below instead. I didn't print any additional ducting beyond the part that attaches to the rear of the printer (yet).
5. [Lentrave's HEPA Filter](https://www.printables.com/model/1022271-qidi-plus-4-filtration-system) - Using this filter cage with the above ducting system.
6. [stew675's Hex Mainboard Cover and Ducting for the CPU fan](https://www.printables.com/model/1146502-qidi-plus-4-hex-mainboard-cover-for-80mm-fan-with/files) - This is an essential upgrade to reduce the operating noise of the printer.
7. [stew675's Filament cutter cable clip](https://www.printables.com/model/1261260-filament-cutter-cable-clip-for-qidi-plus-4)
8. [spi0003's Poop Chute](https://www.printables.com/model/1024451-qidi-plus4-filament-poop-chute) - I split this in 2 parts and glued together to make it easier to print, and not use supports.
9. [SumofThis's Beacon Cable Chain Routing Clips](https://www.printables.com/model/1444399-beacon-usb-cable-chain-routing-clips-mounts-qidi-p/) - Essential otherwise the cable sags and rubs on the X Axis. Zip ties work, but annoying if you need to make tweaks later.
10. [stew675's Chamber Temperature Probe Mount](https://www.printables.com/model/1339760-qidi-plus-4-chamber-temperature-probe-mount) -
11. [rtwickham's Door Hinge Seal](https://cults3d.com/en/3d-model/tool/plus4-articulating-no-hands-door-hinge-seal) - I opted for this so I didn't have to print a seal in TPU.

## Box Essential Mods

1. [GUNgrel's Box Riser](https://www.printables.com/model/1472311-qidi-box-riser-for-qidi-plus-4) - This isn't the modular box riser, I didn't want to bother with that one. This one has worked great. I did have some issues with Australian options for the metal rods though, so I'm using 6.4mm rods instead of the 8mm and CA glued the beam parts together for added strength. It's held up fine.
2. [R985's Filament Guide](https://www.printables.com/model/1386069-qidi-box-jam-resistant-filament-guide) - Not exactly essential, but nice for keeping the PTFE tubing tidy.
3. [GoodHope's Filament Guide Bracket](https://www.printables.com/model/1408927-bowden-tube-support-qidi-plus-4-bracket-for-qidi-b) - Nice addon for the mod above.
4. [GoodHope's Box Hub Enclosure](https://www.printables.com/model/1408451-qidi-box-hub-enclosure-for-jam-resistant-filament) - For the above. Also not exactly essential, but nice.
5. [Colsifer's Spool and Hub Bracket](https://www.printables.com/model/1396483-qidi-plus-4-spool-holder-and-qidi-box-bracket) - I had a side-mount spool mod, but removed it in favour of storing the non-box printer's spools inside a cheap filament dryer.

## Non-Essential Mods

Just nice-to-haves:

1. [osamutake's Gridfinity Tool Holder](https://www.printables.com/model/1345575-qidi-plus4-tool-holder-gridfinity-compatible-1x5) - This was handy in the beginning, but now I have so many other tools I've moved them into better storage.
2. [Sergie's Cell Spool Winder](https://makerworld.com/en/models/561571-cell-spool-winder) - I had some poorly wound spools of filament - this was useful for re-winding them to fix them.
3. [stew675's Super-light Rear Toolhead Cover](https://www.printables.com/model/1156164-qidi-plus4-super-light-rear-toolhead-cover).
